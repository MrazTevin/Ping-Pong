About

PingPong is a simple game that tests divisibility of numbers.When a user inputs a number, it gives out the expected output The results are displayed repeatedly.The goal of the game is to replace numbers divisible by 3, 5 and 15 using either a ping,pong or pingpong


Instructions

Input any number eg 5.The number should be able to diplay the following results.

  • Ping if the number is divisible by 3
  • Pong if the number is divisble by 5
  • Pingpong if the number is divisible by 15
